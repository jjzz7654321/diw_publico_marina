Tipos de Selectores

  • Selectores simples
  • Selectores múltiples
  • Selectores combinados

Selectores simples

  • Tag: por ejemplo un parrrafo (p) o un heading (h1).
  • Identificadores; un identificador de un elemento (el id de un elemento de HTML)
  • Clases: los elementos de HTML tiene un atributo clase que nos permite añadir una o más clases de CSS

Selectores múltiples

Selectores separados por comas. Ejemplo:

article, aside, section {
                backgroud-color: dimgrey;
            }
